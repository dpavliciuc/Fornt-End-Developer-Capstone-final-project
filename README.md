# Little Lemon Booking App

This Booking App was created as the final capstone project of the **Meta Front-End Developer Certification**.

**About**: Little Lemon is a family-owned Mediterranean restaurant that blends traditional recipes with a modern twist. Our goal is to provide our customers with a unique dining experience that will take them on a culinary journey through the Mediterranean.

**Instructions Received**: To create a modern responsive Front-end booking app feature for the Little Lemon which is used to make booking reservations.

---

# Setup and Evaluation

```s
# Run in the Terminal
git clone https://github.com/dpavliciuc/Coursera-Capstone-Booking-Project.git folder

# Install Dependencies
npm install

# Launch app in Browser
npm start

# Run Tests
npm test

# Run Tests with Coverage
npm test:cv
```
